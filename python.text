__init__.py ファイルは、Pythonのパッケージ（ディレクトリをモジュールとして扱うための仕組み）を示すための特別なファイルです。このファイルが存在するディレクトリは、Pythonがパッケージとして扱い、そのディレクトリ内のモジュールやサブパッケージをインポートすることができるようになります。

import: 別のPythonファイルの全体を取り込みます。
from: 別のPythonファイルの一部を取り込みます。

# 辞書の作成
my_dict = {'name': 'John', 'age': 30, 'city': 'New York'}

# keys() メソッドを使用してキーを取得
keys_list = my_dict.keys()

# 結果を出力
print(keys_list)
keys() メソッドは辞書内のキーを取得し、それをビューオブジェクト（dict_keys）として返します。ビューオブジェクトはセットと似たような振る舞いをしますが、リストのように直接インデックスでアクセスすることはできません。ただし、ビューオブジェクトをリストに変換することはできます。

f"Hello, {name}!" は、Python 3.6以降で導入されたf-string（フォーマット文字列）と呼ばれる機能の一部です。f-stringを使用すると、文字列内で変数や式を埋め込むことが簡単になります。
